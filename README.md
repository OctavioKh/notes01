# Notes App

Simple project made with NextJS and NodeJS

## CLONE REPOSITORY
``` git clone https://github.com/OctavioKh/notes01.git```


## Notes
## Install dependencies 

```
cd notas01
npm install

```
## Run development mode
```
npm run dev 
```

## Run server 
On a new terminal
```
cd notas01
cd notes-backend
npm run start
```
